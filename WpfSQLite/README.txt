PROYECTO REALIZADO CON VISUAL STUDIO 2015 COMMUNITY

PAra AnyCPU

Con librerias CLOSEDXML Y OPENXML

Para logs usamos NLOG (desde consola se instalo: Install-Package NLog.Config -Version 4.5.6)

Para que la aplicacion funcione tiene que estar en un grupo de dominio de windows (preguntar a atorres) y el equipo añadido a una regla del firewall para poder conectar la base de datos
